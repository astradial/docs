WhatsApp templates let you create pre-approved message templates for sending automated WhatsApp messages through your campaigns and flows.

## Template list

The dashboard shows all your templates in a table:

| Column | Description |
|--------|-------------|
| **Name** | Template name |
| **Language** | Language code (e.g., EN) |
| **Number** | Integrated phone number |
| **Date** | Creation date |

## Creating a template

<Steps>
  <Step title="Prepare your template">
    Get your template payload as a cURL command or raw JSON from your WhatsApp Business provider.
  </Step>
  <Step title="Paste the payload">
    Click **Create New Template** and paste the full cURL command or JSON. The system automatically extracts the JSON payload.
  </Step>
  <Step title="Edit components">
    After creation, open the template to edit component values for body, header, and buttons.
  </Step>
</Steps>

## Template components

Templates support the following component types:

| Component | Types |
|-----------|-------|
| **Header** | Image, Video, Document |
| **Body** | Text with variable placeholders |
| **Buttons** | URL buttons |

## CRM variable insertion

When editing template components, use the CRM field dropdown to insert lead data dynamically. Available fields include:

- `phone_number`
- `email`
- Any custom fields defined in your lead management settings

<Warning>
  Do not leave placeholder values like `<...>` in your templates. Replace all placeholders with actual values or CRM field references before saving.
</Warning>

## Template actions

| Action | Description |
|--------|-------------|
| **Edit** | Modify component values |
| **Copy ID** | Copy template identifier to clipboard |
| **Delete** | Permanently remove the template |

<Note>
  Template name, language, and namespace are set by your WhatsApp provider and cannot be edited in Astra Dial.
</Note>
