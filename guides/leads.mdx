Lead management centralizes all your contacts in one place. Create leads manually, import in bulk from Excel, configure custom fields, and assign leads to automation flows.

## Viewing leads

The lead table displays:

- Name and phone number
- Email
- Custom fields (user-configured)
- Timestamp
- Action buttons

Use the search bar to find leads by phone number.

## Creating leads

### Manual creation

Click **Create Lead** to open the form. Enter the phone number, name, and any custom fields you've configured.

### Bulk import

Upload an Excel file (.xlsx) to import multiple leads at once.

<Steps>
  <Step title="Click the import button">
    Select your Excel file from the file picker.
  </Step>
  <Step title="Map columns">
    Map the columns from your spreadsheet to lead fields (phone number, name, custom fields).
  </Step>
  <Step title="Review and import">
    Confirm the import. Progress and count are shown during processing.
  </Step>
</Steps>

## Custom fields

Define additional fields beyond the standard name, phone, and email. Navigate to field management to:

- Add new custom fields with labels and field names
- Edit existing fields
- Delete fields you no longer need

Custom fields appear in the lead creation form, table columns, and CSV exports.

## Assigning leads to flows

Use the dropdown on a lead to assign it to an available automation flow (see [Astra Engage](/guides/astra-engage)). This triggers the scheduled tasks in that flow for the lead.

## Exporting leads

Click **Download** to export your leads as a CSV file. You can filter by date range before exporting.
