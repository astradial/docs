Campaigns let you run automated outreach at scale. Create a campaign, select a bot type, and let the system handle calls for you.

## Campaign dashboard

The dashboard shows all your campaigns as cards with:

- Campaign name and type
- Status (active or inactive)
- Statistics: total calls, conversion rate, success rate, failed calls
- Creation and last updated dates

<Info>
  You can have up to **5 campaigns** at a time.
</Info>

## Creating a campaign

<Steps>
  <Step title="Enter campaign details">
    Provide a campaign name and configuration.
  </Step>
  <Step title="Select bot type">
    Choose the bot that powers your campaign:

    | Bot type | Description |
    |----------|-------------|
    | **Astra GenAI** | Conversational AI for natural interactions |
    | **Astra Lite** | Lightweight NLP for structured call flows |
    | **Astra Transcribe** | Audio playback with speech-to-text |
  </Step>
  <Step title="Review and confirm">
    Review your configuration and confirm to create the campaign.
  </Step>
</Steps>

## Campaign actions

| Action | Description |
|--------|-------------|
| **Enable/Disable** | Toggle campaign active status |
| **Update statistics** | Refresh campaign metrics |
| **Upload audio** | Add audio files (for Astra Transcribe campaigns) |
| **Download results** | Export campaign results |
| **Delete** | Remove the campaign |
