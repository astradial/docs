## What is live monitoring?

Live monitoring lets you see phone calls as they happen, right now, in real time. Think of it like a control room where you can watch every active conversation in your organisation. You can see who is on a call, how long they have been talking, and even jump in if needed.

## Viewing active calls

<Steps>
  <Step title="Open the live calls view">
    In the sidebar, go to **Monitor** and click **Calls**. At the top of the page, you will see active calls that are happening right now.
  </Step>
  <Step title="See the details">
    For each active call, you can see:
    - **Who is calling** -- the caller's number or extension
    - **Who they are talking to** -- the agent or extension that answered
    - **Duration** -- how long the call has been going on
    - **Status** -- whether the call is ringing, connected, or on hold
  </Step>
</Steps>

<Info>
  The live calls view updates automatically. You do not need to refresh the page -- new calls appear and finished calls disappear on their own.
</Info>

## Actions you can take

While a call is active, you can take actions on it directly from the dashboard.

### Transfer

Move the call to a different person or extension. This is useful when a caller needs to speak with someone else.

### Hold

Put the call on hold. The caller will hear hold music while they wait. You can take the call off hold at any time.

### Hangup

End the call immediately. Use this if a call needs to be disconnected.

<Warning>
  The **Hangup** button ends the call for everyone. Make sure you really want to end the call before clicking it. There is no undo.
</Warning>

## Supervisor features

If you are a supervisor (Owner, Admin, or Manager), you have special monitoring abilities. These let you listen to and participate in calls without the caller seeing your extension in the call list.

### Spy (silent listen)

Listen to a call without anyone knowing. Neither the agent nor the caller can hear you. This is useful for quality monitoring -- you can hear exactly how your team handles calls.

### Whisper

Listen to the call and **talk to your agent** without the caller hearing you. The agent can hear your coaching, but the caller only hears the agent. This is perfect for training -- you can guide a new team member through a difficult call in real time.

### Barge

Jump into the call so that **everyone can hear you** -- the agent and the caller. This turns the call into a three-way conversation. Use this when you need to step in and help directly.

<Tip>
  Start with **Spy** to understand what is happening, then switch to **Whisper** if the agent needs help, and only **Barge** if you need to speak to the caller directly.
</Tip>

## Who can use live monitoring?

| Feature       | Owner | Admin | Manager | Agent |
| ------------- | ----- | ----- | ------- | ----- |
| View live calls | Yes | Yes   | Yes     | No    |
| Transfer      | Yes   | Yes   | Yes     | No    |
| Hold          | Yes   | Yes   | Yes     | No    |
| Hangup        | Yes   | Yes   | Yes     | No    |
| Spy           | Yes   | Yes   | Yes     | No    |
| Whisper       | Yes   | Yes   | Yes     | No    |
| Barge         | Yes   | Yes   | Yes     | No    |

<Note>
  Agents cannot access the live monitoring view. They can only see their own active call through their softphone app. Supervisory features like spy, whisper, and barge are available to Owners, Admins, and Managers only.
</Note>
