## My Numbers tab

The **My Numbers** tab is your home base for all phone numbers in your organization. To get there, go to the left sidebar, click **Deploy**, then click **Phone Numbers**. You land on the My Numbers tab by default.

Each number in the list shows five things:

| Column | What it means |
|---|---|
| **Number** | The phone number itself (e.g. +91 80 4567 1234) |
| **Description** | A label you or your admin set (e.g. "Main office line") |
| **Routing** | The routing type — Extension, Queue, IVR, AI Agent, or External |
| **Destination** | Where calls go (e.g. extension 1001, Support queue) |
| **Status** | Whether the number is currently working or not |

## Number statuses

Your numbers will show one of these statuses:

- **Active** — The number is live and working. Calls to this number are being routed.
- **Inactive** — The number is paused. Calls to this number will not go through.

<Info>
  If a number shows as Inactive and you are not sure why, contact your organization's admin.
</Info>

## Change routing on a number

You can update how a number routes calls at any time.

<Steps>
  <Step title="Find the number">
    On the **My Numbers** tab, find the number you want to change.
  </Step>

  <Step title="Open the menu">
    Click the **three-dot menu** (⋯) on the right side of that number's row.
  </Step>

  <Step title="Configure routing">
    Click **Configure Routing**.

    Pick a new routing type and destination, then click **Save**. The change takes effect immediately.

    For a full walkthrough of every routing option, see [Configure call routing](/phone-numbers/configure-routing).
  </Step>
</Steps>

## Cancel a pending request

If you requested a number but have not received approval yet, you can cancel the request.

<Steps>
  <Step title="Go to Pending Requests">
    On the **My Numbers** tab, look for the **Pending Requests** section. This shows all numbers you have requested that are still waiting for admin approval.
  </Step>

  <Step title="Cancel the request">
    Click the **Cancel** button next to the pending number.

    The request is removed and the number goes back to the marketplace for others to request.
  </Step>
</Steps>

<Note>
  You can only cancel requests that are still pending. Once an admin has approved a number, it cannot be cancelled from this page.
</Note>
