---
title: "Leads (Contacts)"
description: "Track sales prospects through your pipeline using Kanban boards or list views. Drag leads between stages as they progress."
---

The **Leads** page lets you manage contacts and sales prospects. You can view them as a **Kanban board** (drag-and-drop cards) or a **list** (table view).

## Two view modes

Toggle between views using the tabs at the top of the page:

- **Kanban** — Visual pipeline with columns for each stage. Drag cards to move leads between stages.
- **List** — Traditional table with sortable columns.

## Default lead pipeline stages

| Stage | Meaning |
|-------|---------|
| **New** | Just added, not yet contacted |
| **Contacted** | You have reached out to them |
| **Qualified** | Confirmed interest and budget |
| **Converted** | They became a customer |
| **Lost** | They chose not to proceed |

<Tip>
  You can customize these stages. Go to **Customize** → **Pipelines** → **Lead Pipeline** to add, rename, reorder, or remove stages.
</Tip>

## Add a lead

<Steps>
  <Step title="Click Add Lead">
    Click the **Add Lead** button (plus icon).
  </Step>
  <Step title="Fill in the details">
    | Field | Required | Description |
    |-------|----------|------------|
    | **First Name** | Yes | Contact's first name |
    | **Last Name** | No | Contact's last name |
    | **Email** | No | Email address |
    | **Phone** | No | Phone number |
    | **Job Title** | No | Their role at the company |
    | **Lead Source** | No | Where this lead came from (see below) |
    | **Status** | No | Which pipeline stage (defaults to "New") |
    | **Company** | No | Link to an existing company |
    | **Notes** | No | Any additional notes |
  </Step>
  <Step title="Click Create">
    The lead appears on the Kanban board in the selected stage column.
  </Step>
</Steps>

## Lead sources

When adding a lead, you can track where they came from:

| Source | Description |
|--------|------------|
| Website | Came through your website |
| Phone | Called your phone number |
| Referral | Referred by someone |
| Social | Found you on social media |
| Advertisement | Responded to an ad |
| Cold Call | You reached out first |
| Event | Met at an event or conference |
| Other | Any other source |

## Move leads through the pipeline

### Using Kanban (drag-and-drop)

1. Find the lead card in its current stage column
2. Click and drag it to a new stage column
3. Drop it — the status updates immediately

If the update fails (for example, due to a network issue), the card automatically moves back to its previous position.

### Using the edit dialog

1. Click the three-dot menu on a lead card or table row
2. Click **Edit**
3. Change the **Status** dropdown to the new stage
4. Click **Update**

## Assign leads to team members

1. Click the three-dot menu on a lead
2. Click **Assign**
3. Select a team member
4. The assigned person's name appears as a badge on the lead card

## Search

Use the search bar to filter leads by first name, last name, email, or phone number. Both Kanban and list views are filtered.

## Kanban card details

Each card on the Kanban board shows:
- Lead's full name
- Company name (if linked)
- Lead source badge
- Assigned user badge
- Phone number
